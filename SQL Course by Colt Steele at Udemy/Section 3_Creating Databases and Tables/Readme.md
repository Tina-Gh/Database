## 1. Code: Creating Databases

- Start the CLI:

  mysql-ctl cli; 

- List available databases:

  show databases; 

- The general command for creating a database:

  CREATE DATABASE database_name; 

- A specific example:

  CREATE DATABASE soap_store;
